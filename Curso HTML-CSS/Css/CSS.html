DropShadow

A função text-dropshadow contém 4 elementos.
O 1 px é sobre o quão longe sombra deve ficar para esquerda ou direita.
Depois o 2 px é sobre o quão longe deve ficar do topo ou da bas, que deve cair.
3px é opcional, mas especifica a quantidade de blur.
4px é sobre a cor da sombra.

Pseudo Elementos

Pseudo elementos agem como um código extra.
.a:first-line{ } -> Neste podemos definir qual será a alteração da primeira linha.
.a:first-letter{ } -> Neste podemos definir qual será a alteração da primeira letra.
.a:link{ } -> Neste definiremos como será quando o link não tiver sido acessado ainda.
.a:visited{ } Neste definiremos como será quando o link já tiver sido visitado.

Pseudo classes que mudam a aparência do elemento com a interação do usuário:
input.a:hover -> Quando o mouse estiver em cima, algo na aparência vai mudar.
input.a:active
