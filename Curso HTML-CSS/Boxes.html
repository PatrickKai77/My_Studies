Cada Box tem três valores que podem ser alterados. Border, Margin & Padding
Border: Separa uma ponta de uma box da outra.
Margin: Fica fora da ponta da borda. Podemos definir um widht para margin, assim criamos um gap entre as duas bordas adjacentes de uma caixa.
Padding: É o espaço entre a borda de uma box e qualquer conteúdo dentro dela.
