Estrutura básica de uma tabela.

PAra uma tabela ser iniciada usaremos o elemento table.
Dentro dele deve haver alguns atributos como o tr, que é responsável por indicar o começo de cada linha.

A seguir temos o td, que fica dentro do tr, ele é responsável por criar a célula.

Exemplo

Cabeçalho da Tabela, pode ser definido um titulo da coluna, ou linha.

Para isso usamos th scope="col" ou "row"

As vezes vamos querer espalhar a celulla, tipo que juntar q nem no excel, pra isso usamos colspan="numero de celulas"

Espaçamento e tamanho nas tabelas

Grande parte desses tipos de códigos foram repostos pelo CSS, permitindo uma personalização mais sofisticada.

Podemos também criar espaçamento entre as colnas, linhas e células. Para isto usamos cellpadding(para adicionar espaço dentro de cada célula.
Já o cellspacing é necessário para criar espaço entre cada célula.
